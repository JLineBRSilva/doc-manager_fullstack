# Backend

> Servidor do sistema _**doc-manager_fullstack**_.

## ⌨ SCRIPTS PRINCIPAIS

- ### ( 1 ) Executar última _migration_

    ```bash
    npm run prisma-generate
    ```

- ### ( 2 ) Rodar o servidor

    ```bash
    npm run server
    ```

- ### ( Alternativo ) Executar e abrir o Prisma Studio
    
    ```bash
    npm run studio
    ```

## ⚒️ 🌟 TECNOLOGIAS & FERRAMENTAS USADAS

- [Node.js](https://nodejs.org/en/)
- [Fastify](https://fastify.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Prisma ORM](https://www.prisma.io/docs/prisma-orm/quickstart/postgresql)
- [Cors](https://www.npmjs.com/package/@fastify/cors)
- [PostgreSQL](https://www.postgresql.org/)
- [Postman](https://www.postman.com/)
